# Portfolio — Enyell Duarte (Nokx1z)

Portfolio personal de **Enyell Duarte**, desarrollador backend con enfoque en sistemas escalables y mantenibles. Construido con Next.js 16, TypeScript y Tailwind CSS v4.

## Tecnologías

- **Framework:** Next.js 16 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS v4 + shadcn/ui
- **Animaciones:** Framer Motion
- **Tema:** next-themes (modo claro/oscuro)
- **Íconos:** Lucide React, React Icons
- **Fuentes:** Noto Serif JP, Geist Mono, JetBrains Mono
- **Paquetería:** pnpm

## Características

- Diseño con estética japonesa (kanji, tipografía vertical)
- Alternancia de tema claro/oscuro con animación de transición
- Scroll animado con Framer Motion
- Secciones: Hero, Sobre mí, Tecnologías, Experiencia, Proyectos, Contacto
- CV descargable
- Analytics con Vercel

## Scripts

```bash
pnpm dev        # Inicia servidor de desarrollo
pnpm build      # Construye para producción
pnpm start      # Inicia servidor de producción
pnpm lint       # Ejecuta ESLint
```

## Despliegue

Optimizado para [Vercel](https://vercel.com). Conecta el repositorio y Vercel detectará la configuración automáticamente.
